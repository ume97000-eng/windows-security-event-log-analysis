# windows-security-event-log-analysis
Windows Security Event Log Analysis using Event Viewer (Event IDs 4624, 4672, and 4720)

## Overview
This project demonstrates the analysis of Windows Security Event Logs using Event Viewer. The objective was to identify and understand important security-related events generated by the Windows operating system.

## Events Analyzed

### Event ID 4624 – Successful Logon
This event indicates that a user account successfully logged into the system. It is useful for monitoring authentication activities and detecting suspicious logon attempts.

### Event ID 4672 – Special Privileges Assigned
This event is generated when special administrative privileges are assigned to a new logon session. It helps identify privileged account activity.

### Event ID 4720 – User Account Created
This event records the creation of a new user account on the system. Monitoring this event helps detect unauthorized account creation.
